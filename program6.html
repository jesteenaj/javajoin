Java Program 6: Bitwise operations

This is a basic Java program that prints result of bitwise operations to the console.


class Bitwise {
	public static void main(String args[]) {
		int first = 12;
		int second =25;
		int result;
		int third =2;
		int fourth=8;
		result =first & second;
		System.out.println(result);
		result =first | second;
		System.out.println(result);
		
		result =first | second;
		System.out.println(result);
		
		result =third <<3;
		System.out.println(result);
		
		result =third >> 2;
		System.out.println(result);
	}
}
