Java Program 5: Logical operations

This is a basic Java program that prints result of logical operations to the console.

        class LogicalOPeration {
            public static void main(String args[]) {
                
                System.out.println((5>3) && (8>5));
                System.out.println((5>3) && (8<5));
        
                System.out.println((5<3) || (8>5));
                System.out.println((5>3) || (8<5));
                System.out.println((5<3) || (8<5));
            
                System.out.println(!(5 == 3));
                System.out.println(!(5 > 3));
        
            }
        }
