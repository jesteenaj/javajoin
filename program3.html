Java Program 3: Arithemetic operations

This is a basic Java program that prints result of all operations to the console.

        import java.util.Scanner;
        class ArithmeticAssignment {
            public static void main(String args[]) {
                int first;
                int second;
                Scanner sc=new Scanner(System.in);
                System.out.println("Enter First number");
                first=sc.nextInt();
                System.out.println("Enter Second number");
                second=sc.nextInt();
                System.out.println("Addition="+(first +second));
                System.out.println("Sub= "+(first -second));
                System.out.println("Divison= "+(first /second));
                System.out.println("Product= "+(first *second));
            }
        }
