Java Program 4: Relational operations

This is a basic Java program that prints result of relational operations to the console.

        class RelationalOPerations {
            public static void main(String args[]) {
                int first=10;
                int second=15;
                System.out.println(first == second);
                System.out.println(first != second);
                System.out.println(first > second);
                System.out.println(first >=second);
                System.out.println(first < second);
                System.out.println(first <= second);
            }
        }
