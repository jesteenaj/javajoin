Java Program 7: conditional Operator

This is a basic Java program that prints result of conditional Operator to the console.


        class conditionalOperator {
            public static void main(String args[]) {
            int first;
            int second;
            first=10;
            second=(first==1)?20:30;
            System.out.println("Value of second = "+second);
            
            second=(first==10)?50:20;
            System.out.println("Value of second = "+second);
            }
        }
